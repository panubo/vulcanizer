# Vulcanizer

[VulcanD](http://vulcand.io) frontend updater for [EtcD](https://github.com/coreos/etcd).

## Usage

```
docker run --rm -e ETCDCTL_PEERS=http://10.1.15.122:2379 vulcanizer --host foo --service-name foo-master
```
